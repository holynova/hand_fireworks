# 🎆 3D 手势粒子文字特效

一个基于 Three.js 和 MediaPipe 的交互式 3D 粒子文字特效，通过手势控制粒子的聚散效果。

## ✨ 特性

- 🖐️ **手势识别**：使用 MediaPipe 实时识别手势
- 🎨 **3D 粒子效果**：金币质感的粒子系统
- 🎆 **背景特效**：节日烟花和流星效果
- 🔄 **文字切换**：通过张开手掌切换不同文字
- ⛶ **全屏模式**：支持全屏沉浸式体验

## 🚀 快速开始

### 在线体验

访问 [GitHub Pages](https://your-username.github.io/hand_fireworks/)

### 本地运行

```bash
npm install
npm start
```

然后在浏览器中打开 http://localhost:8080

## 🎮 使用说明

1. 点击"开始体验"按钮，允许摄像头权限
2. **握拳 ✊**：粒子爆发散开
3. **张开手掌 👋**：粒子聚合成文字
4. 自定义文字：点击右上角"重新输入文字"按钮

## 🛠️ 技术栈

- Three.js - 3D 渲染
- MediaPipe - 手势识别
- WebGL - 着色器效果
